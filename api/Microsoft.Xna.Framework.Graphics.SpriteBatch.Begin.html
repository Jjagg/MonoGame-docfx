.SpriteBatch.Begin(SpriteSortMode, BlendState, SamplerState, DepthStencilState, RasterizerState, Effect, Nullable<Matrix>) Method

Begins a new sprite and text batch with the specified render state.

public void Begin(SpriteSortMode sortMode = SpriteSortMode.Deferred, BlendState blendState = null, SamplerState samplerState = null, DepthStencilState depthStencilState = null, RasterizerState rasterizerState = null, Effect effect = null, Matrix? transformMatrix = default(Matrix? ))
Parameters
sortMode

The drawing order for sprite and text drawing. by default.
blendState

State of the blending. Uses if null.
samplerState

State of the sampler. Uses if null.
depthStencilState

State of the depth-stencil buffer. Uses if null.
rasterizerState

State of the rasterization. Uses if null.
effect

A custom to override the default sprite effect. Uses default sprite effect if null.
<> transformMatrix

An optional matrix used to transform the sprite geometry. Uses if null.
