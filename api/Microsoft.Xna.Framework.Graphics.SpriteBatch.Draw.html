.SpriteBatch.Draw(Texture2D, Nullable<Vector2>, Nullable<Rectangle>, Nullable<Rectangle>, Nullable<Vector2>, Single, Nullable<Vector2>, Nullable<Color>, SpriteEffects, Single) Method

Submit a sprite for drawing in the current batch.

[Obsolete("In future versions this method can be removed.")]
public void Draw(Texture2D texture, Vector2? position = default(Vector2? ), Rectangle? destinationRectangle = default(Rectangle? ), Rectangle? sourceRectangle = default(Rectangle? ), Vector2? origin = default(Vector2? ), float rotation = 0F, Vector2? scale = default(Vector2? ), Color? color = default(Color? ), SpriteEffects effects = SpriteEffects.None, float layerDepth = 0F)
Attributes
System.ObsoleteAttribute
Parameters
texture

A texture.
<> position

The drawing location on screen or null if destinationRectangle
<> destinationRectangle

The drawing bounds on screen or null if position
<> sourceRectangle

An optional region on the texture which will be rendered. If null - draws full texture.
<> origin

An optional center of rotation. Uses if null.
rotation

An optional rotation of this sprite. 0 by default.
<> scale

An optional scale vector. Uses if null.
<> color

An optional color mask. Uses if null.
effects

The optional drawing modificators. by default.
layerDepth

An optional depth of the layer of this sprite. 0 by default.
Exceptions

Throwns if both position and destinationRectangle been used.
Remarks

This overload uses optional parameters. This overload requires only one of position and destinationRectangle been used.
