    Help and Support

    This section will provide help and support for MonoGame.

    Help

    If you wish to learn how to use MonoGame, please checkout our the Tutorials. If you want to find an answer to a more specific problem, you can ask it on our Community page.

    Bugs and New Feature Requests

    If you find a bug or have a feature request, please open a new issue. Before opening any issue, please search for existing issues and read the Issue Guidelines.
