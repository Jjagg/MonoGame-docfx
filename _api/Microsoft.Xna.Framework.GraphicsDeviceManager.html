    SynchronizeWithVerticalRetrace

    Indicates the desire for vsync when presenting the back buffer.

    Declaration
    public bool SynchronizeWithVerticalRetrace { get; set; }
    Property Value
    Type Description
    System.Boolean

    Remarks
